About LootJunkie

LootJunkie is an Overwatch loot box simulator where you can open Overwatch loot boxes for free!

How does it work?

Using community data we have a weighted randomization set up to simulate as if you are opening a new loot box everytime.

The current averages are as follows

  • Uncommon: 59%
  • Rare: 32%
  • Epic: 7%
  • Legendary: 2%

Please note that all of the items are rounded and as in game you are guaranteed an item of Rare quality or higher per box.
